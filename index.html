      }

¿Qué hay de Google Analytics en Firefox OS?

No podemos usar el snippet por culpa del Content Security Policy. Usemos "Measurement Protocol".

      $.post( 'http://www.google-analytics.com/collect',
        { v: 1,
          tid: 'UA-264323398-4',
          dp: nombre_pagina,
          t: 'pageview',
          ul: navigator.language
        }
      )

Preprocesador CoffeeScript

  • Lenguaje que compila en CoffeeScript
  • Podemos olvidarnos de las llaves, hay que indentar
  • Olvidarnos de los ";", no necesitamos poner "var".
  • Return implicito en la última línea
  • Funciones y bucles de una sola línea
  • Cambiar el uso del contexto this. por @
  • function() por ->

Hola Mundo JavaScript

      var mensaje = "Hola Mundo";
      var mostrarMensaje = function( mensaje ) {
        alert( mensaje );
      }
      mostrarMensaje( mensaje );

Hola Mundo en CoffeeScript

      mensaje = "Hola Mundo"
      mostrarMensaje = ( mensaje ) -> alert( mensaje )
      mostrarMensaje( mensaje )

Hola Mundo JavaScript Compilado

      var mensaje, mostrarMensaje;
      mensaje = "Hola Mundo";
      mostrarMensaje = function(mensaje) {
        return alert(mensaje);
      };
      mostrarMensaje( mensaje );

Ventajas de CoffeeScript

  • It's just Javascript. (compilar)
  • Escribe menos, haz más. (menos líneas, menos bugs)
  • Código mucho más legible y limpio
  • Compatible con librerías JavaScript ya existentes.
  • Basado en la sintaxis de Ruby y Python

Empresas como Dropbox y AirBnB lo usan.

Desventajas de CoffeeScript

  • No es perfecto. Sensible a los espacios y las tabulaciones
  • El compilador da poca información al detectar un error
  • Hacer debugging puede ser molesto
  • Que el texto sea más legible es discutible
  • Hay mucha gente que lo odia (a veces injustificadamente)
